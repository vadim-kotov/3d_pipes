Задание № 2 ScreenSaver
Выполнил: Котов Вадим Владимирович 328 группа

Установка библиотек:
sudo apt-get install libglfw3-dev
sudo apt-get install libglm-dev
sudo apt-get install libassimp-dev

Сборка проекта (в корневой папке):
mkdir build
cd build
cmake ..
make

Управление камерой:

Камера повораивается вокруг центра сцены
W/S - поворот вокруг горизонтальной оси
A/D - поворот вокруг вертикальной оси
R/F - поворот вокруг своей оси

Esc - Выход

Реализовано: 
- Базовая часть: (10 баллов)
- Аналог скринсейвера для Windows 3D Pipes (1 - 2 балла)
- Управление камерой (1 балл)
